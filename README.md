## 📝 Descrição

Este projeto visa desenvolver uma IA para o jogo Nim utilizando o algoritmo Alpha-Beta Pruning. A IA avalia os estados do jogo com o Nim-Sum e escolhe a melhor jogada de forma otimizada.
